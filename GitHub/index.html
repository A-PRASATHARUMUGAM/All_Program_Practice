1. Setup & Configuration

git config --global user.name "Your Name"

  • Sets your name for commits.
  • Use when: Setting up Git for the first time.

git config --global user.email "you@example.com"

  • Sets your email for commits.
  • Use when: Configuring Git identity.

git config --list

  • Shows all current Git configurations.
  • Use when: You want to verify your setup

2. Repository Initialization

git config --global init.defaultBranch main

git init

  • Initializes a new Git repository in the current folder.
  • Use when: Starting a new project.

git clone "repo_url"

  • Copies an existing repository to your local system.
  • Use when: You want to work on an existing remote repo (e.g., from GitHub).

3. Basic Workflow

git status

  • Shows modified, staged, and untracked files.
  • Use when: You want to check what changes have been made before committing.

git add (filename)

  • Stages a specific file for commit.
  • Use when: You want to include a file in the next commit.

git add .

  • Stages all modified and untracked files.
  • Use when: You want to commit all changes.

git commit -m "Your message"

  • Commits staged changes with a message.
  • Use when:You’ve staged your work and want to save it to the repo.

git log

  • Shows commit history.
  • Use when:Shows commit history.

git show (commit_id)

  • Displays details of a specific commit.
  • Use when:You want to inspect what changed in a commit.

4. Branching

git branch

  • Lists all branches.
  • Use when: You want to see available branches.

git branch (branch_name)

  • Creates a new branch.
  • Use when: You want to start working on a new feature or fix.

git checkout (branch_name)

  • Switches to the specified branch.
  • Use when: You want to move to another branch.

git checkout -b (branch_name)

  • Creates and switches to a new branch.
  • Use when:You want to quickly create and work on a new branch.

git merge (branch_name)

  • Merges changes from another branch into the current one.
  • Use when:You’ve finished work on a branch and want to bring it into main.

git branch -d (branch_name)

  • Deletes a branch.
  • Use when:You’ve merged a branch and no longer need it.
